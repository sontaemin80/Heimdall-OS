**Heimdall-OS**는 Taemin Games 제국의 모든 서비스와 지능형 수호자들에게 접속할 수 있는 차세대 통합 포탈입니다. 단순한 런처를 넘어, **Windows와 macOS** 환경 위에 완벽한 글래스모피즘(Glassmorphism) 기반의 디지털 제국을 구현합니다.

## ✨ Key Features

- **💎 Elite Aesthetics**: 최신 디자인 트렌드를 반영한 투명한 글래스모피즘 UI와 부드러운 애니메이션.

- **🧠 Guardian Intelligence**: Shadow Brain 및 전 제국 수호자들과의 실시간 음성/채팅 연결.
- **⚡ Desktop Native Power**: Flutter 기반의 고성능 **Windows·macOS(Apple Silicon)** 네이티브 엔진으로 최상의 반응성 제공.
- **🌍 Global Access**: 전 세계 어디서나 제국의 자산과 데이터에 즉시 동기화.

## 🚀 Getting Started

Heimdall-OS를 설치하여 제국의 일원이 되십시오. 가장 쉬운 방법은 공식 다운로드 센터
**[taemingames.com/download](https://taemingames.com/download)** 에서 본인 OS에 맞는 빌드를 받는 것입니다.

또는 **[Releases](https://github.com/sontaemin80/Heimdall-OS/releases)** 페이지에서 직접 받을 수 있습니다.

### 🪟 Windows
1. **[Releases](https://github.com/sontaemin80/Heimdall-OS/releases)** 페이지에서 최신 `HEIMDALL_Setup_vX.Y.Z.exe` 인스톨러를 다운로드합니다.
2. 인스톨러를 실행하여 설치를 완료합니다.

### 🍎 macOS (Apple Silicon)
1. **[Releases](https://github.com/sontaemin80/Heimdall-OS/releases)** 페이지에서 `-macos` 태그의 `HEIMDALL_vX.Y.Z.dmg` 를 다운로드합니다. (예: `v3.4.80-macos`)
2. `.dmg` 를 열고 **HEIMDALL.app** 을 Applications 폴더로 드래그합니다.
3. 첫 실행 시 Gatekeeper 경고가 뜨면 우클릭 → **열기** 로 실행합니다. (macOS 12.0+ / M1·M2·M3 권장)

설치 후 제국 계정으로 로그인하여 황실의 권능을 경험하십시오.
